## Essential Concepts

### tmux Hierarchy
```
tmux Server (one per user)
  ↓
Sessions (independent workspaces - like projects)
  ↓
Windows (like browser tabs - within a session)
  ↓
Panes (split screens - within a window)
```

### Key Terminology
- **Prefix Key**: `Ctrl+B` - Press this first, release, then press command key
- **Session**: Independent workspace that persists even if you disconnect
- **Window**: Like a tab in a browser - part of a session
- **Pane**: Split screen within a window - view multiple terminals at once
- **Detach**: Leave session running in background (can reattach later)
- **Attach**: Reconnect to a running session

### Status Bar Guide
```
[session-name] 0:window-name* 1:another-window-
                ↑                ↑
            current window    other windows
```
- `*` = current window
- `-` = last accessed window
- Green bar at bottom = you're IN tmux

## Session Management

### Creating Sessions

```bash
# Create new session with auto-generated name
tmux

# Create new session with specific name
tmux new -s frontend

# Create session in specific directory
tmux new -s work -c ~/dev/project

# Create detached session (runs in background)
tmux new -d -s backend

# Create session with first window named
tmux new -s dev -n editor
```

### Listing Sessions

```bash
# List all running sessions
tmux ls
tmux list-sessions

# Inside tmux: Show session tree
Ctrl+B, S
```

### Attaching to Sessions

```bash
# Attach to most recent session
tmux attach
tmux a

# Attach to specific session
tmux attach -t frontend
tmux a -t frontend

# Attach and detach others (force single connection)
tmux attach -t frontend -d

# Create new session or attach if exists
tmux new -A -s work
```

### Switching Sessions

```bash
# Inside tmux:
Ctrl+B, S              # Interactive session picker
Ctrl+B, (              # Previous session
Ctrl+B, )              # Next session
Ctrl+B, L              # Last session (toggle)

# From command line:
tmux switch -t backend
```

### Renaming Sessions

```bash
# Inside tmux:
Ctrl+B, $              # Rename current session

# From command line:
tmux rename-session -t old-name new-name
```

### Detaching from Sessions

```bash
# Inside tmux:
Ctrl+B, D              # Detach from current session
Ctrl+B, D, D           # Choose which session to detach from (if multiple)

# From command line (detach others):
tmux detach -s frontend
```

### Killing Sessions

```bash
# Kill specific session
tmux kill-session -t frontend

# Kill all sessions except current
tmux kill-session -a

# Kill all sessions except specific one
tmux kill-session -a -t frontend

# Inside tmux: Kill current session
Ctrl+B, :
kill-session
```

## Window Management

### Creating Windows

```bash
# Inside tmux:
Ctrl+B, C              # Create new window
Ctrl+B, &              # Kill current window (with confirmation)

# From command line:
tmux new-window -t frontend -n "api-work"
tmux new-window -t frontend:3 -n "testing"
```

### Switching Windows

```bash
Ctrl+B, 0-9            # Jump to window 0-9
Ctrl+B, N              # Next window
Ctrl+B, P              # Previous window
Ctrl+B, L              # Last accessed window (toggle)
Ctrl+B, W              # Interactive window list (arrow keys to select)
Ctrl+B, '              # Jump to window by index (prompts for number)
Ctrl+B, F              # Find window by name
```

### Renaming Windows

```bash
# Inside tmux:
Ctrl+B, ,              # Rename current window

# From command line:
tmux rename-window -t frontend:0 "main"
tmux rename-window "auth-feature"
```

### Closing Windows

```bash
exit                   # Graceful exit (recommended)
Ctrl+D                 # EOF signal (closes window)
Ctrl+B, &              # Force kill window (with confirmation)
tmux kill-window -t frontend:1
```

### Moving/Reordering Windows

```bash
# Inside tmux:
Ctrl+B, .              # Move window to new index (prompts)

# Swap windows
tmux swap-window -s 2 -t 1
tmux swap-window -t 0

# Move window to different session
tmux move-window -s frontend:1 -t backend:3
```

## Pane Management

### Creating Panes (Splitting)

```bash
Ctrl+B, %              # Split vertically (left | right)
Ctrl+B, "              # Split horizontally (top / bottom)
Ctrl+B, !              # Convert pane to window

# Create specific layout (from command line):
tmux split-window -h   # Horizontal split
tmux split-window -v   # Vertical split
tmux split-window -h -c "#{pane_current_path}"  # Split, keep directory
```

### Switching Panes

```bash
Ctrl+B, Arrow keys     # Navigate to pane in direction
Ctrl+B, O              # Cycle through panes
Ctrl+B, ;              # Toggle between current and last pane
Ctrl+B, Q              # Show pane numbers (type number to jump)
Ctrl+B, Q, 0-9         # Jump to pane by number
```

### Resizing Panes

```bash
Ctrl+B, Ctrl+Arrow     # Resize in arrow direction (hold Ctrl)
Ctrl+B, Alt+Arrow      # Resize in arrow direction (5 cells at a time)
Ctrl+B, :
resize-pane -D 5       # Down 5 lines
resize-pane -U 5       # Up 5 lines
resize-pane -L 5       # Left 5 cells
resize-pane -R 5       # Right 5 cells
resize-pane -x 80      # Set width to 80 cells
resize-pane -y 20      # Set height to 20 lines
```

### Pane Layouts

```bash
Ctrl+B, Space          # Cycle through preset layouts
Ctrl+B, Alt+1          # Even horizontal layout
Ctrl+B, Alt+2          # Even vertical layout
Ctrl+B, Alt+3          # Main horizontal layout
Ctrl+B, Alt+4          # Main vertical layout
Ctrl+B, Alt+5          # Tiled layout
```

### Moving/Swapping Panes

```bash
Ctrl+B, {              # Swap with previous pane
Ctrl+B, }              # Swap with next pane
Ctrl+B, Ctrl+O         # Rotate panes forward
Ctrl+B, Alt+O          # Rotate panes backward
```

### Closing Panes

```bash
exit                   # Graceful exit (recommended)
Ctrl+D                 # EOF signal (closes pane)
Ctrl+B, X              # Kill current pane (with confirmation)
```

### Synchronizing Panes

```bash
# Type same command in all panes simultaneously
Ctrl+B, :
setw synchronize-panes on

# Turn off
Ctrl+B, :
setw synchronize-panes off

# Toggle
Ctrl+B, :
setw synchronize-panes
```

## Copy Mode & Scrolling

### Entering Copy Mode

```bash
Ctrl+B, [              # Enter copy mode (scroll/search mode)
Ctrl+B, Page Up        # Enter copy mode and scroll up one page
```

### Navigation in Copy Mode

```bash
# Scrolling
Arrow keys             # Move one line at a time
Page Up / Page Down    # Move one page at a time
Ctrl+U                 # Scroll half page up
Ctrl+D                 # Scroll half page down
Ctrl+B                 # Scroll full page up
Ctrl+F                 # Scroll full page down

# Jumping
g                      # Jump to top of history
G                      # Jump to bottom (latest output)
0                      # Jump to start of line
$                      # Jump to end of line
w                      # Jump forward one word
b                      # Jump backward one word
```

### Searching in Copy Mode

```bash
/                      # Search forward (type text, press Enter)
?                      # Search backward
n                      # Next search result
N                      # Previous search result
```

### Exiting Copy Mode

```bash
q                      # Exit copy mode
Escape                 # Exit copy mode
```

## Copy & Paste

### Selecting and Copying Text

```bash
# Vi mode (default):
Ctrl+B, [              # Enter copy mode
Space                  # Start selection
Arrow keys             # Expand selection
Enter                  # Copy and exit copy mode

# Rectangle selection:
Ctrl+B, [
Ctrl+V or V            # Start rectangle selection
Arrow keys             # Expand selection
Enter                  # Copy
```

### Pasting Text

```bash
Ctrl+B, ]              # Paste most recent buffer
Ctrl+B, =              # Choose which buffer to paste (interactive)
```

### Managing Buffers

```bash
# List paste buffers
Ctrl+B, #
tmux list-buffers

# Show buffer content
tmux show-buffer -b 0

# Save buffer to file
tmux save-buffer -b 0 ~/output.txt

# Load file into buffer
tmux load-buffer -b 0 ~/input.txt

# Delete buffer
tmux delete-buffer -b 0
```

## Configuration

### Mouse Support

```bash
# Enable mouse (inside tmux):
Ctrl+B, :
set -g mouse on

# Disable mouse:
Ctrl+B, :
set -g mouse off

# Permanent (add to ~/.tmux.conf):
set -g mouse on
```

**Mouse Features:**
- Click to switch panes
- Drag borders to resize
- Scroll wheel to scroll history
- Double-click to select word
- Triple-click to select line

### Vi or Emacs Mode

```bash
# Set Vi mode (inside tmux):
Ctrl+B, :
set -g mode-keys vi

# Set Emacs mode:
Ctrl+B, :
set -g mode-keys emacs

# Permanent (add to ~/.tmux.conf):
set -g mode-keys vi
```

### History Limit

```bash
# Increase scrollback (inside tmux):
Ctrl+B, :
set-option -g history-limit 50000

# Permanent (add to ~/.tmux.conf):
set-option -g history-limit 50000
```

### Status Bar

```bash
# Toggle status bar:
Ctrl+B, :
set -g status off     # Hide
set -g status on      # Show

# Customize (add to ~/.tmux.conf):
set -g status-position top          # Move to top
set -g status-interval 5            # Update every 5 seconds
set -g status-left-length 40        # Longer session name
set -g status-right "%H:%M %d-%b-%y"  # Custom right side
```

### Color Scheme

```bash
# Add to ~/.tmux.conf:
set -g default-terminal "screen-256color"
set -g pane-border-style fg=colour238
set -g pane-active-border-style fg=colour51
set -g status-style bg=colour235,fg=colour136
```

### Reload Configuration

```bash
# Inside tmux:
Ctrl+B, :
source-file ~/.tmux.conf

# From command line:
tmux source-file ~/.tmux.conf
```

### Sample ~/.tmux.conf

```bash
# ~/.tmux.conf - Basic Configuration

# Enable mouse support
set -g mouse on

# Increase scrollback buffer
set -g history-limit 50000

# Vi mode
set -g mode-keys vi

# Better colors
set -g default-terminal "screen-256color"

# Status bar at top
set -g status-position top

# Longer status bar
set -g status-left-length 40

# Renumber windows when one is closed
set -g renumber-windows on

# Start windows and panes at 1 (not 0)
set -g base-index 1
setw -g pane-base-index 1

# Better pane borders
set -g pane-border-style fg=colour238
set -g pane-active-border-style fg=colour51

# Split panes using | and -
bind | split-window -h
bind - split-window -v
unbind '"'
unbind %

# Reload config
bind r source-file ~/.tmux.conf \; display "Config reloaded!"

# Quick pane switching with Alt+Arrow (no prefix needed)
bind -n M-Left select-pane -L
bind -n M-Right select-pane -R
bind -n M-Up select-pane -U
bind -n M-Down select-pane -D
```

## Workflow Examples

### Example 1: Frontend Development Setup

```bash
# Create session for frontend work
tmux new -s frontend -c ~/dev/wildlife-watcher-mobile-app

# Window 0: Editor
# Enable mouse
Ctrl+B, : set -g mouse on

# Window 1: Development server
Ctrl+B, C
Ctrl+B, , "dev-server"
npm start

# Window 2: Testing
Ctrl+B, C
Ctrl+B, , "testing"
npm test

# Window 3: Git
Ctrl+B, C
Ctrl+B, , "git"
git status

# Detach and continue working
Ctrl+B, D
```

### Example 2: Monitoring Layout (Panes)

```bash
# Create monitoring session
tmux new -s monitor

# Enable mouse
Ctrl+B, : set -g mouse on

# Split into 4 panes
Ctrl+B, %              # Split vertical
Ctrl+B, Left           # Go to left pane
Ctrl+B, "              # Split horizontal
Ctrl+B, Right Right    # Go to right pane
Ctrl+B, "              # Split horizontal

# Result:
# ┌─────────┬─────────┐
# │  Top    │  Top    │
# │  Left   │  Right  │
# ├─────────┼─────────┤
# │ Bottom  │ Bottom  │
# │  Left   │  Right  │
# └─────────┴─────────┘

# Fill with monitoring tools
# Pane 0 (top-left): htop
# Pane 1 (bottom-left): docker stats
# Pane 2 (top-right): tail -f logs
# Pane 3 (bottom-right): free -h; watch -n 5 free -h
```

### Example 3: Multiple Projects

```bash
# Frontend session
tmux new -d -s frontend -c ~/dev/frontend
tmux send-keys -t frontend 'claude code' Enter

# Backend session
tmux new -d -s backend -c ~/dev/backend
tmux send-keys -t backend 'claude code' Enter

# Database session
tmux new -d -s db -c ~/dev/database
tmux send-keys -t db 'psql' Enter

# List all sessions
tmux ls

# Attach to frontend
tmux attach -t frontend

# Switch to backend
Ctrl+B, S (select backend)

# Or from command line
tmux switch -t backend
```

### Example 4: Pair Programming

```bash
# Person A creates session
tmux new -s pair-programming

# Person B attaches to same session
tmux attach -t pair-programming

# Both see and control the same session!
# When Person A types, Person B sees it instantly
# When Person B types, Person A sees it instantly
```

### Example 5: Automated Startup Script

```bash
#!/bin/bash
# ~/start-work.sh

SESSION="work"

# Create session if doesn't exist
if ! tmux has-session -t $SESSION 2>/dev/null; then
    # Create session
    tmux new -d -s $SESSION -c ~/dev/project

    # Window 0: Editor
    tmux rename-window -t $SESSION:0 'editor'
    tmux send-keys -t $SESSION:0 'code .' Enter

    # Window 1: Server
    tmux new-window -t $SESSION -c ~/dev/project -n 'server'
    tmux send-keys -t $SESSION:1 'npm start' Enter

    # Window 2: Git
    tmux new-window -t $SESSION -c ~/dev/project -n 'git'
    tmux send-keys -t $SESSION:2 'git status' Enter

    # Window 3: Testing
    tmux new-window -t $SESSION -c ~/dev/project -n 'testing'
    tmux send-keys -t $SESSION:3 'npm run test:watch' Enter
fi

# Attach to session
tmux attach -t $SESSION
```

```bash
chmod +x ~/start-work.sh
~/start-work.sh  # One command to start entire workspace!
```

## Troubleshooting

### Common Issues

#### "tmux: command not found"
```bash
# Install tmux
sudo apt update && sudo apt install tmux -y
```

#### "error connecting to /tmp/tmux-1000/default"
```bash
# Normal! No sessions running. Create one:
tmux new -s work
```

#### "sessions should be nested with care"
```bash
# You're trying to start tmux inside tmux!
# Exit current session first:
exit
# Then create new session
tmux new -s another
```

#### "duplicate session: frontend"
```bash
# Session already exists
# Option 1: Attach to it
tmux attach -t frontend

# Option 2: Kill and recreate
tmux kill-session -t frontend
tmux new -s frontend

# Option 3: Use different name
tmux new -s frontend-v2
```

#### Can't Scroll with Mouse
```bash
# Enable mouse support:
Ctrl+B, :
set -g mouse on

# Make permanent:
echo "set -g mouse on" >> ~/.tmux.conf
```

#### Lost Sessions After Reboot
```bash
# Expected behavior!
# tmux sessions only survive disconnects, not full reboots
# Use startup scripts to recreate workspace quickly
```

#### Ctrl+B Not Working
```bash
# Check you're in tmux (green bar at bottom?)
# If yes, verify prefix key:
Ctrl+B, :
show-option -g prefix

# Make sure you're doing: Press Ctrl+B, RELEASE, then press command key
# Not holding Ctrl+B while pressing command key
```

### Diagnostic Commands

```bash
# Check if in tmux
echo $TMUX
# Empty = not in tmux
# Path shown = in tmux

# List all sessions with details
tmux list-sessions -F "#{session_name}: #{session_windows} windows (created #{session_created_string})"

# Show all keybindings
Ctrl+B, ?

# Show current settings
tmux show-options -g

# Debug mode (logs to file)
tmux -vv new -s debug
# Check /tmp/tmux-*.log
```

## Advanced Tips

### Custom Key Bindings

```bash
# Add to ~/.tmux.conf:

# Split panes with | and - (easier to remember)
bind | split-window -h
bind - split-window -v

# Reload config with r
bind r source-file ~/.tmux.conf \; display "Reloaded!"

# Switch panes with Alt+Arrow (no prefix needed!)
bind -n M-Left select-pane -L
bind -n M-Right select-pane -R
bind -n M-Up select-pane -U
bind -n M-Down select-pane -D

# Quick window switching with Shift+Arrow
bind -n S-Left previous-window
bind -n S-Right next-window
```

### Session Management Scripts

```bash
# Kill all sessions except current
tmux kill-session -a

# Kill all sessions
tmux kill-server

# List sessions with more detail
tmux list-sessions -F "#{session_name}: #{session_windows} windows, #{session_attached} attached"
```

### Copy Mode Vi Keybindings

```bash
# Add to ~/.tmux.conf for vi-like copy mode:
set -g mode-keys vi
bind -T copy-mode-vi v send -X begin-selection
bind -T copy-mode-vi y send -X copy-selection-and-cancel
bind -T copy-mode-vi C-v send -X rectangle-toggle
```

### Plugin Manager (TPM)

```bash
# Install TPM (Tmux Plugin Manager)
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm

# Add to ~/.tmux.conf:
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'tmux-plugins/tmux-resurrect'  # Save/restore sessions
set -g @plugin 'tmux-plugins/tmux-continuum'  # Auto-save sessions

# Initialize TPM (keep at bottom of .tmux.conf)
run '~/.tmux/plugins/tpm/tpm'

# Install plugins: Ctrl+B, I
# Update plugins: Ctrl+B, U
# Uninstall: Ctrl+B, Alt+U
```
