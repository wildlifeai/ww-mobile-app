# Active Coordination Directory

Current coordination activities and in-progress cross-project work.

## Purpose

This directory contains:
- Active coordination tasks requiring attention
- In-progress backend-mobile integration work
- Current schema change coordination
- Ongoing cross-repo feature development

## Workflow

### Adding Active Items
Place files here when:
- Starting new cross-project coordination
- Backend schema change requires mobile attention
- Cross-repo feature requires parallel development
- Urgent coordination issue needs tracking

### Completing Active Items
When coordination completes:
1. Update completion status in the file
2. Move to appropriate archive subdirectory:
   - Date-based: `../archive/2025-{month}/`
   - Type-based: `../archive/completion-reports/`

## File Naming Convention

Use descriptive names with dates:
- `schema-change-{feature}-{YYYY-MM-DD}.md`
- `coordination-{task-id}-{description}.md`
- `urgent-{priority}-{summary}.md`

## Status Tracking

Each active file should include:
- **Status**: In Progress / Blocked / Waiting
- **Owner**: Mobile team / Backend team / Both
- **Started**: Date coordination began
- **Expected Completion**: Target date
- **Blockers**: Any dependencies or issues

## Related Directories

- **Protocols** (`../protocols/`) - Coordination protocol documentation
- **Templates** (`../templates/`) - Message templates
- **Archive** (`../archive/`) - Completed coordination activities
