# Utilities

This folder contains **utility files** and prompts for system operations.

## Contents

- **IMPLEMENTATION-CONTINUATION-PROMPT.md** - Instructions for continuing implementation work
  - Context for resuming work
  - Current status summary
  - Next steps guidance

- **x-project-implementation-prompt.md** - Short implementation prompt
  - Quick reference prompt
  - Cross-project context

## Purpose

Small utility files that don't fit into other categories:
- Not user-facing guides
- Not design documentation
- Not completion reports
- Not active metrics

These are **operational aids** for development work.

## Usage

Reference these files when:
- Resuming implementation work after a break
- Need quick context on cross-project coordination
- Setting up new development sessions
