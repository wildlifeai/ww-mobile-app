# Backend Coordination Protocol

Protocols for coordinating work between mobile and backend teams.

## Contents

- **BACKEND-REPOSITORY-ANALYSIS.md** - Comprehensive analysis of backend repository structure and patterns
- **cross-project-coordination-reference.md** - Quick reference for coordination workflows

## Key Concepts

- **Schema Change Coordination**: Backend migrations → mobile type updates
- **API Integration**: Backend endpoints → mobile RTK Query slices
- **Cross-Repo Features**: Parallel development across repositories
- **Release Coordination**: Synchronized deployments and versioning

## Coordination Channels

- Shared coordination hub: `~/dev/wildlifeai/cross-project-coordination/`
- File-based messages with automatic routing
- Git hooks for schema change detection
- Status updates and task requests
