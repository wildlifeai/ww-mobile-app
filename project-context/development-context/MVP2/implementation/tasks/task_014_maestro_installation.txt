```bash
./install-maestro-wsl2.sh

# This script handles:
# - Java 17+ installation verification
# - Maestro CLI installation via official script
# - PATH configuration for WSL2
# - Installation verification
# - WSL2-specific setup notes
```

**Manual Installation Steps (Alternative)**
```bash
# 1. Verify/Install Java 17+
java -version
# If < 17: sudo apt update && sudo apt install openjdk-17-jdk

# 2. Install Maestro CLI
curl -Ls "https://get.maestro.mobile.dev" | bash

# 3. Configure PATH
echo 'export PATH="$HOME/.maestro/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

# 4. Verify Installation
maestro --version
```
