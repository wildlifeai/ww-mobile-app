PARALLEL DEVELOPMENT STREAM A - Task 14: Project Details & Administration

This task completes the project management system by providing comprehensive project details and administrative capabilities for Wildlife Watcher projects.

## Implementation Requirements:

### 14.1: Project Details Screen
- Create `src/navigation/screens/ProjectDetailsScreen.tsx`
- Comprehensive project information display:
  - **Header Section**: Project name, description, creation date, owner info
  - **Project Settings**: Sampling design, website, privacy settings
  - **Project Statistics**: Total deployments, active deployments, team size
  - **Member Section**: Team members with roles (from Task 13)
  - **Activity Feed**: Recent project activities and updates
  - **Deployment History**: List of recent deployments with status
- Editable fields for project admins:
  - Project description, sampling design, website URL
  - Privacy settings (private/public toggle)
  - Project settings (bait usage, monitoring marked)
- Real-time updates using Supabase subscriptions

### 14.2: Project Administration Tools
- **Project Settings Management**:
  - Update project metadata and configuration
  - Change project privacy and access settings
  - Modify project-specific settings (bait, monitoring)
- **Data Management**:
  - Export project data (deployments, member activity)
  - Generate project reports (deployment success rates, team performance)
  - Data cleanup and archival tools
- **Project Lifecycle Management**:
  - Archive inactive projects (preserves data, removes from active lists)
  - Delete projects (admin only, with confirmation and cascading cleanup)
  - Transfer project ownership between admins

### 14.3: Analytics and Reporting
- **Project Analytics Dashboard**:
  - Deployment success metrics
  - Team member activity statistics
  - Device usage patterns
  - Location-based deployment insights
- **Export Capabilities**:
  - CSV export of deployment data
  - PDF project summary reports
  - JSON data export for external analysis
  - Integration with external analytics tools

### 14.4: Advanced Features
- **Project Templates**: Save project configurations as templates for future use
- **Bulk Operations**: Manage multiple deployments simultaneously
- **Integration Points**: Connect with external wildlife monitoring systems
- **Audit Trail**: Track all project modifications and administrative actions
