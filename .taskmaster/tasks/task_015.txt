## Wizard Flow:
1. **Project Selection** → Choose project and name deployment
2. **New Project** → (Optional) Create project if needed
3. **Device Discovery** → Scan and select BLE device
4. **Configuration** → Set location and capture method
5. **Camera Preview** → Validate camera positioning
6. **Final Setup** → Add details and create deployment
