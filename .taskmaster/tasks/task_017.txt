PARALLEL DEVELOPMENT STREAM B - Task 17: Deployment Status & Management

This task implements the comprehensive deployment management system that enables users to monitor, track, and manage all their wildlife monitoring deployments.

## Implementation Requirements:

### 17.1: Deployments List Screen
- Create `src/navigation/screens/DeploymentsScreen.tsx`
- **Tabbed Interface**: Organized tabs for Active / Ended / All deployments
- **Deployment Cards**: Rich information display for each deployment:
  - Project name, deployment name, device identifier
  - Location information with coordinates/address
  - Duration (for active) or total duration (for ended)
  - Battery level and SD card usage indicators
  - Last updated timestamp and connectivity status
- **Pull-to-Refresh**: Manual refresh to get latest deployment status
- **Search and Filter**: Find deployments by name, project, location, or status
- **Performance Optimization**: Efficient list rendering for large datasets

### 17.2: Deployment Details Screen
- Create `src/navigation/screens/DeploymentDetailsScreen.tsx`
- **Comprehensive Information Display**:
  - Deployment metadata (name, project, dates, duration)
  - Device information and current status
  - Location details with map integration
  - Configuration details (capture method, settings)
  - Status history and timeline
  - Associated files and media
- **Quick Actions**:
  - End deployment (if active)
  - View on map
  - Share deployment details
  - Export deployment data

### 17.3: Real-Time Status Update System
- **Supabase Real-time Subscriptions**: Live updates for deployment changes
- **Battery Level Monitoring**: Real-time battery status with low battery alerts
- **Storage Monitoring**: SD card usage tracking with storage warnings
- **Connectivity Status**: Device online/offline status indicators
- **Alert System**: Push notifications for critical status changes
- **Background Updates**: Status updates without blocking UI interactions

### 17.4: Status Indicators & Visual Feedback
- **Status Chips**: Color-coded status indicators (Active, Ended, Issues)
- **Battery Indicators**: Visual battery level with warning states
- **Storage Indicators**: SD card usage with capacity warnings
- **Connectivity Icons**: Real-time connection status
- **Last Updated Timestamps**: Time since last status update
- **Issue Detection**: Automatic detection and flagging of problems
